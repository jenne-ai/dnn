# How to use
Starting point is the file "Call_Algos_File". Depending on Parameter F 
different input data is used.

In the __main__ part die configuration and initialization of the net is given.

The file "Classifier" is supposed to handle different classification algorithms.
By now two are given MLP (Multi Layer Perceptron) and DNN.
The source code of these classifier is given in the files "MultiLayerPerceptron"
and "DNN_flex".
